Important notes

Keep the workflow clean

Use one sheet

The main template sheet is copied once and used throughout the series.

Do not skip the page

The sheet is for doing the work. The episode page explains what to do, why it matters, and how to make the tab work.

Scripts are shared episode by episode

When an episode needs automation, the Apps Script code and setup steps will be shared on that episode page. Apps Script works only inside a native Google Sheet, not inside Excel .xlsx mode.

Keep credentials private

When API credentials are needed, enter them only inside your own copied sheet or Apps Script setup. Never enter private credentials into a public template.
