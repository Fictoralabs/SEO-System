Create the site structure first

SEO starts when the website idea is being planned. The first task is to search the main keyword, study the companies already ranking, and understand what pages the website should have.

Task from the reel

Build the site structure from ranking websites

Task steps

  • Open incognito. Search the main keyword the company should rank for. In the reel example, the keyword is whatsapp automation Dubai.
  • List ranking companies. Record the companies or competitors appearing at the top of Google.
  • Visit each website. Open every competitor website and list the pages they have created.
  • Find the pattern. Use the collected pages to understand how many pages should be created and what the site structure should look like.

Automation note
