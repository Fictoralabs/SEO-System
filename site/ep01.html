Build the site structure from ranking websites

Concept

SEO starts when the idea is being planned and the website is about to be built. The first step is to create the site structure.

To create the site structure, search the main keyword in an incognito tab. In the reel example, the keyword is whatsapp automation Dubai. The companies ranking at the top show what Google already understands for that category.

Open each ranking website and list the pages they have created. The repeated page patterns help decide how many pages are needed and what the website structure should look like.

Task steps

  • Open incognito. Search the main keyword the company should rank for. In the reel example, the keyword is whatsapp automation Dubai.
  • List ranking companies. Record the companies or competitors appearing at the top of Google.
  • Visit each website. Open every competitor website and list the pages they have created.
  • Find the pattern. Use the collected pages to understand how many pages should be created and what the site structure should look like.

Use this sheet tab

01 COMPETITORS

Open the latest master sheet and use this tab for EP01. Enter the competitor list created from the Google search. Use one row per competitor.

Sheet task

  • Fill the ranking page URL, competitor name, and domain.
  • Use the robots.txt and sitemap helper fields when available.
  • Use the sitemap/page output to understand the important pages competitors have created.

Automation note

The robots.txt and sitemap helpers are only there to make page collection easier. The decision still comes from reviewing competitor pages and finding the repeated structure.
